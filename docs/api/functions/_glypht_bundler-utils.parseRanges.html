parseRanges

  • parseRanges(ranges: string): (number | readonly [number, number])[] | null

    Parse a series of non-Unicode ranges or values. You can use this for parsing values to use for instanced variation axes.

    Parsing works similarly to parseUnicodeRanges--you can parse single values or ranges, separated by commas or just spaces. Unlike parseUnicodeRanges, all numbers are decimal. Decimal points are accepted, but "e" syntax is not.

    Parameters

    • ranges: string

      The string of ranges to parse.

    Returns (number | readonly [number, number])[] | null

    An array of numbers, or tuples of numbers for ranges. If the input string contains invalid syntax, returns null.
